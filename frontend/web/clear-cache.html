🧹 Clear Flutter App Cache

Instructions to clear cache:

  1. Open Chrome Developer Tools (F12)
  2. Go to Application tab
  3. In left sidebar, click Local Storage → http://localhost:8080
  4. Right-click and select Clear
  5. Go to Session Storage → http://localhost:8080
  6. Right-click and select Clear
  7. Go to IndexedDB and clear any Firebase related data
  8. Click buttons below to clear additional storage
